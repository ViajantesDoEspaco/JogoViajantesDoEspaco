# Viajantes Do Espaco
Jogo implementado em C e SDL2 para a disciplina de Jogos

## 1. Qual o nome do projeto?
Viajantes do Espaço

## 2. Qual o tipo do projeto?
É um jogo 2d no estilo space invaders. A implementação inicial será baseada no jogo original.

## 3. Descrição do Projeto
O projeto consiste na criação de um jogo no estilo space invaders, onde o jogador controla uma nave no espaço e atira nos inimigos, destruindo suas naves. Quando um determinado numero de naves é destruída o jogador ganha power ups que podem ser velocidade, intensidade dos tiros, aumento de vida, etc
